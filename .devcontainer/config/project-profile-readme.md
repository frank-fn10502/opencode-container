# opencode-dev project profile

You are in the project profile directory. This project may be opened from the host, an IDE, a VM, or the opencode-dev container.

Create or edit project profiles here:

```text
Dockerfile.<profile>
```

For the full Dockerfile rules and examples:

```text
Host:      ~/.opencode-dev-yuta/README.md
Container: /opencode-dev/user/README.md
```

If you are running inside a container or VM, treat the user profile as read-only and do not modify it from here.

If you are running on the host and need to create or edit user profiles, find the user profile directory:

```text
~/.opencode-dev-yuta/
```

With opencode-dev, edit user profiles by opening `opencode-dev` from the host `$HOME` directory.
